# MVP仕様

## 目的

飲食店を保存し、保存済み候補の中から今日行く1店舗を決められるようにする。

## 対象範囲

- 共有URLまたは手動入力から飲食店を保存する
- 基本的な店舗情報を保存する
- 保存済み店舗を表示する
- 訪問済み / 未訪問を管理する
- 行きたい度を設定する
- 基本条件でフィルタする
- ガチャモードで決める
- スワイプモードで候補を絞る

## 対象外

- Googleマップの完全インポート
- Instagramの完全インポート
- 予約フロー
- ソーシャルフィード
- 高度なAIコンシェルジュ
- 複雑な自動タグ付け
- 大規模なレコメンド基盤

## 成功基準

- ユーザーが候補を素早く保存できる。
- ユーザーがアプリを開き、先に一覧管理をしなくても決定に到達できる。
- ユーザーが訪問後に最低限の評価を記録できる。
